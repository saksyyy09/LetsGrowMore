To-Do List Project
This project, the "To-Do List," is a task management web application developed for the Let's Grow More internship program. It allows users to create and manage their tasks in an organized manner.

Project Overview
The To-Do List project is designed to help users keep track of their tasks and improve productivity. It provides a user-friendly interface where users can create new tasks, mark tasks as complete, and delete tasks when necessary. The application aims to streamline task management and enhance overall efficiency.

Technologies Used
The project was built using the following technologies:

HTML
CSS
JavaScript
